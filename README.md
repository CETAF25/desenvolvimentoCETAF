# CETAF | Advanced Manufacturing & Automation

O **CETAF** (Creativity, Efficiency, Technology, Automation, Future) é um grupo de desenvolvimento técnico dedicado à criação de soluções para a indústria 4.0. Nosso foco reside na integração de hardware, visão computacional e gestão de dados para otimização de processos produtivos.

## 🛠 Áreas de Atuação
* **Sistemas de Automação:** Desenvolvimento de lógica para controladores e integração de sensores.
* **Visão Computacional & IA:** Implementação de triagem automatizada e reconhecimento de padrões.
* **Projetos Mecânicos:** Modelagem 3D e detalhamento técnico de dispositivos industriais.
* **Gestão e Branding:** Padronização técnica de documentação e comunicação visual de projetos.

## 🚀 Portfólio de Projetos
Atualmente, o grupo mantém os seguintes módulos de desenvolvimento:

### [SAS - Sistema Automatizado de Separação]
Uma célula de triagem industrial que utiliza visão computacional para detecção de falhas e separação de peças em tempo real.
* **Status:** Em desenvolvimento (Ciclo 2026.1)
* **Tecnologias:** Arduino, Python, CAD/3D.
